# Assyrian Empire Musuem

This project is a digital archive of the Assyrian Empire.

## Description

This project provides history, figures, and artifacts of the Assyrians.
This project is separated into three front-facing pages, one accessible helper page, and one inaccessible helper page.
The main home page is a scrolling timeline of the Assyrian Empire from nacent years to its fall.
This timeline covers major historical figures of the Empire.
A secondary page is a gallery of artifacts of the Assyrians.
The gallery presents names, categories, descriptions, and pictures.
The gallery uses a dual purpose filter to sort through artifacts by category and name search.
The third page is a donation page.
The donation page holds an HTML form for people willing to donate an artifact from the era and keep the memory of the assyrians alive.
The donation page links to an accessible helper page as the target of the form data.
The helper page has links to the three front-facing pages of the project.
An inaccessible helper page holds artifact information as a multidimensional associative array in a php page.
Lastly the project has a directory of images of artifacts.

### Dependencies

The project needs a computer capable of running PHP.
The project needs a modern browser capable of rendering a large amount of data (pictures).

### Install

The project repository is available at: https://github.com/Exemption9888/Luu_AssyrianEmpire.git
All available files and images are ready to download.

### Execute

The project will be accessible through the web server hosting the files.
The home page will be accessed at https://[your-webserver-directory]/index.php
